# 📚 Jornada de Aprendizado & Desafios de Código

Este repositório é um registro centralizado de todos os exercícios, desafios e projetos práticos que realizei durante meus estudos em Análise e Desenvolvimento de Sistemas (FIAP) e outras plataformas de tecnologia.

## 🎯 Objetivo
Demonstrar minha evolução técnica, organização de raciocínio lógico e a aplicação prática de conceitos de desenvolvimento de software, desde algoritmos básicos até aplicações web completas.

## 🛠️ Tecnologias Exploradas
As soluções aqui presentes abrangem as seguintes tecnologias e conceitos:

* **Front-end:** HTML5, CSS3 (Flexbox/Grid), JavaScript (DOM, ES6+), Bootstrap, jQuery.
* **Back-end:** Lógica de Programação
* **Versionamento:** Git e GitHub.
